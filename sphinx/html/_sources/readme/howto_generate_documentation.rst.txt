------------------------
Generating documentation
------------------------

Checkout current version from repository: https://wrscmi.inside.nsn.com/isource/svnroot/BTS_DOCS_SMALLCELLS/trunk

Windows & Linux
===============

1. Enter the directory with documentation source code.
2. Run ``make html``.

Linux (LinSee server)
=====================

1. Enter the directory with documentation source code.
2. Run ``./build_html_linux.sh``
